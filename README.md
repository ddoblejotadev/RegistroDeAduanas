# Registro de Aduanas

Aplicación escolar para gestionar declaraciones aduaneras (Viajero / Funcionario). Adaptada para funcionar localmente sin depender de Base44.

Instrucciones rápidas:

1. git clone https://github.com/ddoblejotadev/RegistroDeAduanas
2. cd RegistroDeAduanas
3. npm install
4. npm run dev

Cuentas de prueba:
- viajero@test.cl / viajero123
- admin@aduana.cl / admin123

Si deseas eliminar totalmente dependencias de Base44, puedo hacerlo (requiere probar build).
